# UAS_STKI
UAS Sistem Temu Kembali Informasi

## Library
program ini menggunakan Python Library **text-preprocessing**(Tokenizing, Filtering dan Stemming) yaitu:

* [NLTK(Natural Language Toolkit)](https://github.com/nltk/nltk) - Permodelan teks
* [PySastrawi](https://github.com/har07/PySastrawi) - Stemming  bahasa Indonesia

## Implementasi
implementasi TF-IDF menggunakan python dan dataset
* [Source](https://github.com/gitarja/blog) - implementasi TF-IDF

Jika terjadi error pada proses stemming :
pip install sastrawi
atau
pip install pysastrawi
